# Developer Tools

These scripts are for local development and debugging only. Do not import these in production code.

| Script | Purpose | How to run |
|--------|---------|------------|
| `test_scrape_and_summarize.py` | Manually trigger scrapers and test summary generation | `py tools/test_scrape_and_summarize.py` |
| `view_data.py` | Inspect the local SQLite database contents | `py tools/view_data.py` |

Run from the `backend/` directory.
